Welcome to Huggle

Huggle is an anti-vandalism tool for use on Wikipedia and other Wikimedia projects.

In addition to the license shown here, all source code is released into the GPL. However, some images used by the project are available only under the LGPL.

Comments and complaints should be directed to the feedback page. If you want commit access, ask in the same place. Bug tracker lives at bugzilla

Howto checkout source code

Project members

$ git clone git@github.com:benapetr/huggle.git

Anonymous
